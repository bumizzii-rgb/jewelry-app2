📋 วิธีสร้าง Google Apps Script
ขั้นตอน:
1. เปิด Google Sheets ใหม่ → ตั้งชื่อ "Jewelry Database"
2. ไปที่เมนู Extensions → Apps Script
3. ลบโค้ดเดิม แล้ววางโค้ดด้านล่างนี้:


4. กด Deploy → New deployment → Web app
5. Execute as: Me | Who has access: Anyone
6. กด Deploy → คัดลอก URL มาวางด้านล่าง
🔗 URL การเชื่อมต่อ
URL นี้จะถูกบันทึกไว้ในอุปกรณ์นี้
